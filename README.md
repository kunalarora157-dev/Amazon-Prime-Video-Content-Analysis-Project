# Amazon-Prime-Video-Content-Analysis-Project
Amazon Prime Video Content Analysis Project
Amazon Prime Video Content Analysis
Project Summary
This project analyzes Amazon Prime Video's content library to identify trends in genres, ratings, content growth, production countries, and contributor participation.
The analysis uses:
Python
Pandas
NumPy
Matplotlib
Seaborn
Business Problem
Amazon Prime Video hosts thousands of movies and TV shows. Understanding content diversity, ratings, popularity, and regional production patterns helps support content investment and strategic decisions.
Dataset
titles.csv
Contains:
title
show_type
release_year
genres
imdb_score
tmdb_score
runtime
credits.csv
Contains:
actor names
director names
character names
roles
Key Objectives
Analyze Movies vs TV Shows
Identify dominant genres
Analyze content growth over time
Study IMDb and TMDb ratings
Identify top actors and directors
Tools Used
Pandas
NumPy
